The Gametools Java Library

Gametools is a lightweight and simple to use library for creating 2D games in Java.

Some features of this library include:

  • Sprites
  • Image and spritesheet loading
  • Collision handling
  • Keyboard and mouse input
  • Minimal setup code
  • Sprite grouping and automation
  • Animation
